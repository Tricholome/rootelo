# Rootelo

Automated Elo-based leaderboard for Root Digital League.

## 🚀 Live Website
👉 **[View the Leaderboard](https://tricholome.github.io/rootelo)**

## ⚙️ How it works
This project automatically fetches match data from the Root Digital League API, calculates Elo ratings using Python, and updates the website once a day using GitHub Actions.

*Last updated: Automated via GitHub Actions*
